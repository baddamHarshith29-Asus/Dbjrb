# Evaluator Project

This project contains a safe expression evaluator built using Python's AST module. The evaluator can compute a limited set of arithmetic expressions while preventing arbitrary code execution.

## Features
- Supports basic arithmetic operations (addition, subtraction, etc.)
- Allows specific mathematical functions from Python's math library
- Rejects unsupported syntax to ensure safety.

## Installation
To use the evaluator, clone the repository and run the test file.

```bash
python test_evaluator.py
```
